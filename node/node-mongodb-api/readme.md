## Installation
First of all you need Node.js<br>
Download and install 10.x<br>
https://nodejs.org/en/

Next, open terminal in folder 'node-mongodb-api' and run command
```
npm i
```
Done.

## Starting server
Open terminal in folder 'node-mongodb-api' and run command
```
npm run api
```
Done.
